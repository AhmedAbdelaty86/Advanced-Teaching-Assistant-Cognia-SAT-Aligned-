Use this skill when any teacher wants to generate a full Cognia-aligned lesson package for a topic. Triggers include: requests like "create a lesson for", "build my teaching materials", "generate SAT quiz", "make a lesson plan", "create teaching artifacts", "Phase 1 lesson", "Phase 2 analysis", or any mention of lesson planning + SAT/Cognia standards. Also trigger when the user uploads classroom content files (PDFs, Word docs) and asks to build a lesson around them. This skill drives a strict two-phase workflow: Phase 1 produces a Word lesson plan, a Canva presentation, and a SAT-style exit quiz; Phase 2 analyzes student results and generates a skills report + action plan. ALWAYS use this skill for teaching material creation — even if the user only mentions one artifact, the full workflow applies.
